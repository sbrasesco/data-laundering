## n8n Sub-workflow v4

### URL producción
`POST https://automation.aignition.net/webhook/sub-document`

### Archivo
`workflow_subworkflow_v4.json`

### Flujo
```
Webhook → Set Campos de Entrada → Descargar y Preparar Archivo
    ↓ (fan-out)
Leer PDFs1 → Añadir fileName → Extract from File → Normalizar PDF ─┐
Leer OC Map → (parallel)                                            │
Leer Imagenes JPG → Mistral OCR → Normalizar OCR JPG ──────────────┤
Leer Imagenes PNG → Mistral OCR → Normalizar OCR PNG ──────────────┤
                                                                    ↓
                                               Unir todo texto → Inyectar OC Map
                                                                    ↓
                                               Loop Over Items → AI Agent (GPT-4.1-mini)
                                                                    ↓
                                               Parsear JSON IA → Adjuntar Job a filas
                                                                    ↓
                                               ¿Tiene Factura? → Create a row (pdf_job_rows)
                                                                    ↓
                                               Log + Expandir OCs + Confianza baja?
```

### Cambios clave vs v2 (original con ZIP)
- **Entrada**: acepta `file_url` JSON en lugar de binario ZIP
- **Descarga**: `wget -qO` en lugar de `7z x` (n8n container no tiene curl)
- **Execute Command**: usa `={{ "FILE_URL='" + $json.file_url + "' ..." }}` (expresión JS, no inline `{{ }}`)
- **Inyectar OC Map / Parsear JSON IA**: Function node v1 con `$items()` (accede ramas paralelas, Code node v2 no puede)
- **Referencias**: `$('Edit Fields')` → `$('Set Campos de Entrada')`

### Payload del webhook
```json
{
  "job_id": "uuid",
  "organization_id": "uuid",
  "file_url": "https://...",
  "file_type": "pdf",
  "original_filename": "factura.pdf",
  "client_cuit": null,
  "client_name": null,
  "oc_entries": [],
  "input_source": "frontend_upload"
}
```
